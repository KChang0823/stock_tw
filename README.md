# 📈 台灣高股息價值選股器 (Stock Valuation Tool)

基於 **FinMind Data API** 與專業 **四分位階估價模型** 打造的自動化選股工具。本專案能自動推估 EPS、預測股利，並計算出精確的買賣位階。

## ✨ 特色功能
- **動態 EPS 推估**：結合「直接累計」與「成長率對比」兩種模型，產出高低預估區間。
- **配息率比例法**：根據去年配息金額與 EPS 成長比例，精確預測今年現金與股票股利。
- **四分位階估價**：實作「低買入、高買入、低賣出、高賣出」四個關鍵價位，並包含除權修正係數。
- **指標畫布 (Fintech UI)**：採用深色模式、玻璃擬態設計與互動式位階圖表。

## 🚀 快速啟動

### 1. 安裝環境
建議使用 Python 3.9+ 環境：
```bash
pip install -r requirements.txt
```

### 2. 啟動 App
```bash
streamlit run app.py
```

## 🛠 核心架構
- `app.py`: Streamlit 展示層，負責 UI 與視覺化。
- `valuation_engine.py`: 邏輯層，實作所有核心估價公式。
- `data_loader.py`: 資料層，透過 FinMind API 獲取財報與股價數據。
- `邏輯.md`: 專案核心計算邏輯與公式定義說明文件。

## 📊 估價公式參考
詳細公式請參閱 [邏輯.md](./邏輯.md)。核心概念：
- **買入位階**：現金股利 × 16 倍 (對應 6.25% 殖利率)
- **賣出位階**：現金股利 × 32 倍 (對應 3.125% 殖利率)
- **除權修正**：1 + (股票股利 / 10)
