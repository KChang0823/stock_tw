# 高股息估價邏輯說明

## 基本邏輯
這套 Excel 文件的核心目標是透過「預期股利」來反推股票的「合理買賣區間」。

### 1. 核心估價公式：殖利率位階法
此公式結合了現金股利與股票股利對股價位階的影響，產出四個關鍵價位：

*   **買入價 (16 倍位階)**：
    *   **低買入價** = `(零件 C × 16) / (1 + 零件 E / 10)`
    *   **高買入價** = `(零件 D × 16) / (1 + 零件 F / 10)`

*   **賣出價 (32 倍位階)**：
    *   **低賣出價** = `(零件 C × 32) / (1 + 零件 E / 10)`
    *   **高賣出價** = `(零件 D × 32) / (1 + 零件 F / 10)`

*   **代號定義**：
    *   **零件 C / D**：低 / 高 預計現金股利。
    *   **零件 E / F**：低 / 高 預計股票股利。
    *   **分母 (1 + 股票股利/10)**：這是 Excel 中處理股票配息對股價除權影響的修正係數。

---

### 2. 動態 EPS 推估邏輯 (核心技術)
這套邏輯透過比較「實際累計值」與「權重放大值」來得出高、低兩組預估。

*   **推估公式**：
    *   **低預估 EPS** = `MIN( 零件A , 零件B )`
    *   **高預估 EPS** = `MAX( 零件A , 零件B )`

*   **零件拆解**：
    *   **零件 A (直接累計)**：`Q1 + Q2 + Q3 + Q4` (實際已填入的四季 EPS 之和)
    *   **零件 B (EPS 成長推估法)**：
    `B = (今年已公布季 EPS 總和 / 去年同期 EPS 總和) × 去年全年總 EPS`

*   **邏輯核心**：
    *   **低預估**：走保守路線，怕權重放大太樂觀或是實際數值太低。
    *   **高預估**：走樂觀路線，看好成長潛力或權重推算的爆發力。
    *   這兩組數值會分別導致後續產出「低位階價 (便宜價)」與「高位階價 (昂貴價)」。

---

### 3. 預計股利計算 (配息率比例法)
此步驟將 EPS 轉化為實際配息。邏輯是：**假設公司維持與去年相同的配息率 (Payout Ratio)**。

*   **核心公式 (以現金股利為例)**：
    *   **低預計現金股利 (零件 C)** = `去年現金股利 × (今年低預估 EPS / 去年總 EPS)`
    *   **高預計現金股利 (零件 D)** = `去年現金股利 × (今年高預估 EPS / 去年總 EPS)`

*   **核心公式 (以股票股利為例)**：
    *   **低預計股票股利 (零件 E)** = `去年股票股利 × (今年低預估 EPS / 去年總 EPS)`
    *   **高預計股票股利 (零件 F)** = `去年股票股利 × (今年高預估 EPS / 去年總 EPS)`

*   **邏輯核心**：
    *   不再使用固定的歷史平均值，而是以「去年實際發放金額」為基準，根據今年賺錢的能力（EPS）比例進行縮放。

---

### 4. 操作訊號標籤
系統會根據現價落在哪個位階區間來自動給予建議：

| 現價區間 | 顯示訊號 | 意義 |
| :--- | :--- | :--- |
| **現價 < 低買入價** | **買** | 進入強力價值區 |
| **低買入價 ≤ 現價 ≤ 高買入價** | **追蹤** | 進入買進觀察區 |
| **高買入價 < 現價 < 低賣出價** | (不顯示) | 區間震盪，無明顯訊號 |
| **低賣出價 ≤ 現價 ≤ 高賣出價** | **追蹤** | 進入賣出觀察區 |
| **現價 > 高賣出價** | **賣** | 進入獲利了結區 |

---

### 5. 資料來源 (Data Source)
*   目前已從 **FinMind API** 自動抓取近三年的季 EPS、股利發放記錄（現金/股票）以及歷史殖利率，作為計算上述公式的原料。

## 實戰範例演練：以「某高股息龍頭股」為例

假設目前時點為第三季財報剛公布：

### 第一步：推估全年 EPS
*   **已知數據**：
    *   **今年 (2025)**：Q1=2.5, Q2=2.5, Q3=2.5 (累計 7.5), Q4 尚未公布 (假設預填 2.0)
    *   **去年 (2024)**：Q1=2.0, Q2=2.0, Q3=2.0 (同期累計 6.0), 全年總 EPS = 8.0
*   **計算零件**：
    *   **零件 A (直接累計)**：2.5 + 2.5 + 2.5 + 2.0 = 9.5
    *   **零件 B (成長推估)**：(7.5 / 6.0) × 8.0 = 10.0
*   **最終推估**：
    *   **低預估 EPS** = `MIN(9.5, 10.0) = 9.5`
    *   **高預估 EPS** = `MAX(9.5, 10.0) = 10.0`
    > [!NOTE]
    > 若零件 A 與 B 出現分歧（例如 Q4 實際填入較保守或權重推算較樂觀），系統會分別產出高低兩組股利與估價。

### 第二步：計算預計股利 (假設與去年配息比例一致)
*   **已知數據 (去年 2024)**：
    *   去年總 EPS = 8.0
    *   去年現金股利 = 5.0 / 去年股票股利 = 0.5
*   **計算今年低預期 (EPS=9.5)**：
    *   預計現金 = 5.0 × (9.5 / 8.0) = **5.94 元**
    *   預計股票 = 0.5 × (9.5 / 8.0) = **0.59 元**
*   **計算今年高預期 (EPS=10.0)**：
    *   預計現金 = 5.0 × (10.0 / 8.0) = **6.25 元**
    *   預計股票 = 0.5 × (10.0 / 8.0) = **0.625 元**

### 第三步：計算估價位階 (正式產出四個價位)
我們使用前面算出的零件 C, D (現金) 與 E, F (股票)：
*   零件 C=5.94, D=6.25
*   零件 E=0.59, F=0.625

*   **1. 低買入價** = `(5.94 × 16) / (1 + 0.59 / 10)` = `95.04 / 1.059` ≈ **89.75 元**
*   **2. 高買入價** = `(6.25 × 16) / (1 + 0.625 / 10)` = `100.0 / 1.0625` ≈ **94.12 元**
*   **3. 低賣出價** = `(5.94 × 32) / (1 + 0.59 / 10)` = `190.08 / 1.059` ≈ **179.49 元**
*   **4. 高賣出價** = `(6.25 × 32) / (1 + 0.625 / 10)` = `200.0 / 1.0625` ≈ **188.24 元**

### 第四步：判斷訊號
假設目前的四個參考價分別為：**89.75 (低買)**、**94.12 (高買)**、**179.49 (低賣)**、**188.24 (高賣)**。

*   若股價落在 **85 元**：因為 `85 < 89.75` $\rightarrow$ **顯示「買」**。
*   若股價落在 **92 元**：因為 `89.75 < 92 < 94.12` $\rightarrow$ **顯示「追蹤」**。
*   若股價落在 **130 元**：因為 `94.12 < 130 < 179.49` $\rightarrow$ **(不顯示標籤)**。
*   若股價落在 **182 元**：因為 `179.49 < 182 < 188.24` $\rightarrow$ **顯示「追蹤」**。
*   若股價落在 **195 元**：因為 `195 > 188.24` $\rightarrow$ **顯示「賣」**。
